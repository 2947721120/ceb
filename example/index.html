Custom elements

  • Address Selector: an auto-complete widgets based on baconjs helping to select an existing address.
  • List: a simple todo list like widget using handlebarjs.
  • List++: like the ceb-list where handlebarjs has been replaced by incremental-dom.
  • Templator: an extension of the Script element to automatically parsed the element's content (handlebarsjs).
  • Grid: a simple data grid handling filtering, sorting and paging. Data can be get from a memory store or a REST store.
  • Form: a set of custom elements to handle an HTML form and its content

Webapps

  • Todo App: a todo app based on redux, incremental-dom and, obviously, custom elements.
