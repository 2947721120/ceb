ceb's examples

Examples are written in ES6 and compiled into the browser, so according to your device, you will wait for or not.

Sources are hosted in GitHub

Load with AMD, UMD or System

  • AMD modules
  • UMD modules
  • System modules
  • Standalone

Custom elements

  • ceb-address-selector: an auto-complete widgets based on baconjs helping to select an existing address.
  • ceb-list: a simple todo list like widget using handlebarjs.
  • ceb-list-plusplus: like the ceb-list where handlebarjs has been replaced by incremental-dom.
  • ceb-templator: an extension of the Script element to automatically parsed the element's content (handlebarsjs).

Webapp

  • todo-app: a todo app based on redux, incremental-dom and, obviously, custom elements.
