    Custom Element Builder

  • ¶

    ceb is a builder to help the development of Custom Elements.

    • The project is hosted on Github
    • Read the documentation for more details
    • Or the commented source code
    • Every use cases are tested and validated from this test suite
    • The source code is continuously built on Travis
    • The test suite is automatically executed using Sauce Labs
    • The code coverage report is pushed to coveralls

    Compatibilities

    ceb should and will work without dependencies on evergreen browsers. To execute it with none evergreen browsers, you should need of:

    • Obviously Custom Elements polyfill
      • webcomponents-lite.js from webcomponents.org
      • or document-register-element
    • ES5 and some features from ES6 es6-shim

    Downloads

    Distributed files can be found there

    • ceb.min.js minified
    • ceb.legacy.min.js shims, minified

    var builder = ceb().name('a-custom-element');

builder.properties({
    name: {
        attribute: true
    }
}).methods({
    sayHelloTo: function (el, n) {
        return el.name + ' say hello to ' + n + '!';
    }
});

builder.intercept('name', function (next, el, value) {
    next(value.toUpperCase());
}).wrap('sayHelloTo', function (next, el, n) {
    return next(n.toUpperCase());
});

builder.register();

var element = document.createElement('a-custom-element');
element.setAttribute('name', 'I');
element.sayHelloTo('you');

element.name = 'Again, I';
element.sayHelloTo('you');
