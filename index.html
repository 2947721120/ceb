custom-elements-builder

is just a builder, natively scalable and designed for FRP

  • The project is hosted on Github
  • Change logs, coveralls, Travis, Sauce Labs

<ceb/> should and will work without dependencies on evergreen browsers. However for the others, like IE9/IE10, you have to use a Custom Elements polyfill:

  • webcomponents-lite.js from webcomponents.org
  • or document-register-element
